### Fetch patient data from an API, find Jessica Taylor, and display her complete medical information in a 3-column dashboard that matches the Adobe XD design.

### Project Live link :  https://fastidious-cheesecake-bb3a3e.netlify.app/

What Is This Project?
A Patient Health Dashboard built for Coalition Technologies FED Skills Test. It's a single-page web application that doctors use to view patient medical data fetched from a live API.

The URL to the Adobe XD template can be found here - https://xd.adobe.com/view/3f9ab587-7536-4db8-a7dd-64d474e10867-6ac9/?hints=off

## Tech Stack :- 
HTML: Page structure
CSS : Styling and layout
JavaScript: Logic and API calls
Chart.jsBlood pressure graph
REST API:Live patient data

## How The API Works
URL:      https://fedskillstest.coalitiontechnologies.workers.dev
Method:   GET
Auth:     Basic Auth → btoa('coalition:skills-test')
Returns:  Array of 10 patients in JSON format
